# Python DNS Domain to IP Resolution

This Python script demonstrates the Domain Name System (DNS) resolution process. It takes a domain name (like example.com) and translates it into its corresponding IP address, mimicking a crucial step a web browser performs after you click a link. This is how the browser finds the server to request content from.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
3. Optionally, provide a domain name as an argument: `python main.py example.org`
